# Execute in Dev

1. Clone repository
1. Execute:

   ```
   yarn install
   ```

1. Install nest CLI

   ```
   npm i -g @nestjs/cli
   ```

1. Star the database
   ```
   docker-compose up -d
   ```

## Stack

- MongoDB
- NestJS
